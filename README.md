## What I am building next

### GitHub Security Posture Auditor

A Python tool to analyze GitHub repositories and organizations for common security risks:

* missing branch protection
* exposed secrets risk
* weak repository metadata
* missing security policy
* dependency and supply-chain visibility gaps
* missing CI/CD security controls
* misconfigured repository permissions
* lack of SBOM or artifact integrity controls

The goal is to produce a practical security scorecard and actionable remediation report for engineering and security teams.
